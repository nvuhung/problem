# Problems

## Installation

```bash
# install node_modules
$ npm install
```

## Start

Run in _src/index.js_

```bash
# run start
$ npm start

# watch mode
$ npm run watch:start
```

## Unit Test

Run in _test/index.spec.js_

```bash
# run test
$ npm test

# watch mode
$ npm run watch:test
```
